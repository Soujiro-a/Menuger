# Menuger

'매일 점심 시간마다 뭘 먹을까 고민이신가요?'  
'적은 돈으로 만들 수 있는 식사를 찾고 계신가요?'  
'살 쭉쭉 빠지는 다이어트 식단을 없을까?'

당신의 매니저가 되어 당신의 고민을 끝내드리겠습니다.

대한민국의 모든 밥 고민을 없애줄 서비스 `Menuger` 입니다.
