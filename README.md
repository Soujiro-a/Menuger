# Menuger

'오늘 점심 뭐 먹지?'  
'가성비 좋은 식단은 없을까?'  
'새우 알레르기가 있는데... 새우를 쓰지않는 레시피는 없을까?'  
이런 고민을 해보신 적이 있으신가요?  

Menuger를 사용하면 나와 비슷한 고민을 가진 사람이 공유한 레시피와 식단을 찾아볼 수 있습니다.  
더 나아가, 자신만의 메뉴를 직접 구성해보고 사람들과 공유해보세요!  

여러분의 밥 고민을 해결해 줄 서비스 Menuger 입니다.
